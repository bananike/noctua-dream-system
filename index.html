Notes

가격 변동성

램(CORSAIR)·SSD(9100 PRO)는 낸드·D램 시황 급등 직격탄. 실제 구매 시 재확인. 9100 PRO 4TB 최근 130만원대 → 170만원대.

메모리 · 6000 CL30 (AMD 최적)

AMD는 인피니티 패브릭 1:1 동기화가 핵심이라 6000~6400이 스위트스팟. 8000+ 고속은 오히려 손해. 하이닉스 다이 6000 CL30이 EXPO로 안정적. 64GB 용량도 확보.

SSD 방열판 · Gen5 슬롯 배치

9100 PRO 2장 모두 방열판 미포함 선택 · AERO X3D 보드 자체 M.2 방열판 사용(양면 SSD 이중 방열판 간섭 방지). 부팅용 9100 PRO는 M2A_CPU(Gen5 x4 풀속도)에 장착. 단 두 번째 Gen5 슬롯 M2B_CPU는 USB4 컨트롤러와 대역폭 공유 → 장착 시 둘 다 x2로 강등되므로, USB4를 상시 풀로 쓰려면 두 번째 9100 PRO를 칩셋 Gen4 x4 슬롯에 배치(실측 Gen5 x2 ≈ Gen4 x4 수준).

가격 변동성

램(CORSAIR)·SSD(9100 PRO)는 낸드·D램 시황 급등 직격탄. 실제 구매 시 재확인. 9100 PRO 4TB 최근 130만원대 → 170만원대.

메모리 · CUDIMM 8400 (Intel 최적)

Arrow Lake(Core Ultra 200S)는 메모리 컨트롤러가 타일로 분리돼 고속 램에서 이득이 크다. CUDIMM은 클럭 드라이버(CKD)를 내장해 8000+ 고속을 안정화 — 일반 UDIMM보다 여유롭다. 2×24GB(48GB) 8400 CL40 구성.

SSD 방열판 · Gen5 슬롯 배치

9100 PRO 2장 모두 방열판 미포함 선택 · ProArt Z890 보드 자체 M.2 방열판 사용(양면 SSD 이중 방열판 간섭 방지). 부팅용 9100 PRO는 M.2_1(CPU 직결 Gen5 x4 풀속도)에 장착. 단 Z890은 CPU Gen5 M.2가 1개뿐이라 두 번째 9100 PRO는 칩셋 Gen4 x4 슬롯(~7,450MB/s)에 배치 — 체감 차이는 작지만 순차 대역폭은 절반.
